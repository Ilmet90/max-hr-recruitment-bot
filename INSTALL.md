# INSTALL

Подробная установка MAX HR Recruitment Bot на Debian/Ubuntu Server.

## 1. Подготовить сервер

Нужен сервер Debian или Ubuntu с доступом в интернет. Обновите систему:

```bash
sudo apt update
sudo apt upgrade
```

Если `git` ещё не установлен:

```bash
sudo apt install git
```

## 2. Клонировать репозиторий

```bash
git clone https://github.com/Ilmet90/max-hr-recruitment-bot.git
cd max-hr-recruitment-bot
```

## 3. Запустить установщик

```bash
sudo bash install.sh
```

Установщик скопирует проект в `/opt/max-hr-recruitment-bot`, создаст виртуальное окружение, установит зависимости, создаст `.env`, инициализирует SQLite-базу и настроит systemd-службы.

## 4. Ответить на вопросы

Установщик попросит:

- полное название организации;
- краткое название организации;
- вышестоящую организацию;
- регион;
- название бота;
- логин главной web-учётной записи;
- пароль главной web-учётной записи;
- служебный код для команды `/admin`;
- `MAX_BOT_TOKEN`;
- host и port web-админки.

Пароль, служебный код и `MAX_BOT_TOKEN` вводятся обычным видимым текстом. Если пароль или служебный код оставить пустыми, установщик сгенерирует их и покажет один раз в конце установки.

`MAX_BOT_TOKEN` можно оставить пустым. В этом случае web-админка будет установлена и запущена, а службу бота можно будет запустить позже после заполнения токена.

## 5. Если MAX_BOT_TOKEN пока нет

После получения токена откройте:

```bash
sudo nano /opt/max-hr-recruitment-bot/.env
```

Заполните `MAX_BOT_TOKEN`, затем выполните:

```bash
sudo systemctl start max-hr-bot.service
sudo systemctl enable max-hr-bot.service
```

## 6. Открыть web-админку

Откройте:

```text
http://<IP_сервера>:<APP_PORT>
```

Войдите под логином и паролем главной учётной записи.

## 7. VirtualBox и NAT

Если сервер запущен в VirtualBox через NAT, настройте проброс порта. Пример:

```text
host 8010 -> guest 8000
```

Тогда на основной машине открывайте:

```text
http://127.0.0.1:8010
```

Если порт занят, выберите другой свободный host-порт, например `8011` или `8080`.

## 8. Первичная настройка

После входа в web-админку:

1. Откройте «Настройки организации».
2. Проверьте название организации, тексты приветствия, контакты и цвета.
3. Откройте «Настройки» и при необходимости смените служебный код `/admin`.
4. Создайте или импортируйте вакансии.
5. Добавьте условия работы или службы и фотоальбомы.

## 9. Регистрация сотрудников

Сотрудник пишет боту:

```text
/admin <служебный код>
```

После этого главная учётная запись или начальник кадрового подразделения подтверждает заявку в web-админке или через `/staff`.

## 10. Настройка импорта вакансий

1. Войдите в web-админку.
2. Откройте «Вакансии → Импорт с Работа России».
3. Укажите ИНН работодателя или код работодателя на портале.
4. Включите импорт и сохраните настройки.
5. Нажмите «Проверить вакансии».
6. Проверьте предпросмотр.
7. Нажмите «Импортировать найденные вакансии».

## 11. Включить управление обновлениями

Для проверки обновлений, обновления из GitHub и перезапуска служб из web-интерфейса выполните один раз:

```bash
sudo bash /opt/max-hr-recruitment-bot/scripts/setup_maintenance_sudoers.sh
```

Скрипт создаёт ограниченный sudoers-файл только для управления службами проекта.

## 12. Проверить службы

```bash
sudo systemctl status max-hr-admin.service
sudo systemctl status max-hr-bot.service
```

Логи:

```bash
sudo journalctl -u max-hr-admin.service -f
sudo journalctl -u max-hr-bot.service -f
```

## 13. Сделать первый backup

```bash
cd /opt/max-hr-recruitment-bot
sudo bash scripts/backup.sh
```

## 14. Удалить проект

```bash
cd /opt/max-hr-recruitment-bot
sudo bash scripts/uninstall.sh
```

Скрипт спросит, удалять ли данные. Если данные нужны для архива или переноса, не подтверждайте удаление каталога проекта.
